# LingoPitch Frontend

> NeoBrutalism design for international sales training platform

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS (Neobrutalism theme)
- Zustand (state management)
- Framer Motion (animations)
- i18next (multi-language)

## Development

```bash
npm install
npm run dev
```

## Design System

- **Colors**: Primary (orange), Accent (teal), Dark palette
- **Shadows**: Brutal box shadows (8px offset, black border)
- **Typography**: Space Grotesk (display), Inter (body), JetBrains Mono (code)
- **Animations**: Float, slide, scale, shimmer

## Features

- 80+ language support via Lingo.dev
- Real-time cultural context adaptation
- Neobrutalism + Glassmorphism UI
- Smooth parallax scrolling
